Conteúdos Estudados

A disciplina de Desenvolvimento Web costuma ser organizada em ordem de complexidade: primeiro se aprende a estruturar uma página, depois a estilizá-la, e só então a torná-la interativa. Essa ordem faz sentido porque cada tecnologia depende da anterior.

Outro ponto estudado é a responsividade: um site precisa funcionar bem tanto em computadores quanto em celulares e tablets. É por isso que frameworks como o Bootstrap, usado neste projeto, são tão estudados - eles já vêm prontos para se adaptar automaticamente a diferentes tamanhos de tela.

Abaixo está uma lista com os principais tópicos estudados na disciplina. Clique em cada item da lista para ver uma explicação rápida sobre ele.

  • HTML (estrutura da página)
  • CSS (estilo e aparência)
  • JavaScript (interatividade)
  • Frameworks (como o Bootstrap)
  • Responsividade
  • Formulários e validação

Além dos conteúdos técnicos, a disciplina também trabalha boas práticas, como organizar o código em pastas (uma para HTML, outra para CSS, outra para JavaScript - exatamente como neste projeto) e testar o site em diferentes tamanhos de tela antes de finalizar.
