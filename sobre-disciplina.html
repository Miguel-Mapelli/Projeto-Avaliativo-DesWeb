Sobre a Disciplina de Desenvolvimento Web

O que é a disciplina?

Desenvolvimento Web é a disciplina do curso de ADS que ensina a criar sites e sistemas que funcionam pela internet. Diferente de um programa de computador comum, instalado em uma máquina específica, um site criado com tecnologias web pode ser acessado por qualquer pessoa, em qualquer lugar, usando apenas um navegador.

A disciplina costuma ser dividida em três grandes tecnologias que trabalham juntas. O HTML é responsável pela estrutura da página - define o que existe na tela: textos, imagens, botões, listas. O CSS cuida da aparência - cores, espaçamentos, tamanhos de fonte e a forma como os elements se organizam visualmente. Já o JavaScript dá vida à página, permitindo criar interações, como um botão que reage a um clique ou um formulário que valida o que foi digitado.

Além dessas três tecnologias, a disciplina também apresenta frameworks, que são "caixas de ferramentas" prontas que facilitam o trabalho. Um exemplo é o Bootstrap, usado neste próprio site, que já vem com componentes prontos, como menus de navegação e botões estilizados.

Por que essa disciplina é importante?

A internet está presente em praticamente tudo hoje em dia: comprar um produto, acessar o banco, estudar, se comunicar com outras pessoas. Por trás de cada um desses sites existe alguém que entende de Desenvolvimento Web. Por isso, essa disciplina prepara o estudante de ADS para uma das áreas com mais oportunidades de trabalho dentro da tecnologia.

Outro motivo de importância é que Desenvolvimento Web costuma ser uma das primeiras disciplinas em que o estudante consegue ver, na prática, um resultado visual e imediato do código que escreveu. Diferente de outras disciplinas mais abstratas, aqui é possível abrir o navegador e ver a própria página funcionando.

Por fim, o conhecimento de Desenvolvimento Web também serve de base para áreas mais avançadas, como desenvolvimento de aplicativos, sistemas para empresas e lojas virtuais. Quem entende bem o "tripé" HTML, CSS e JavaScript tem facilidade para aprender essas tecnologias mais avançadas depois.
