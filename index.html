Desenvolvimento Web

Este site foi feito como projeto avaliativo da disciplina de Desenvolvimento Web, do curso de Análise e Desenvolvimento de Sistemas (ADS) do Instituto Federal de Catanduva. Desenvolvimento Web é a minha disciplina favorita do curso, e por isso escolhi ela como tema principal deste site.

De forma resumida, Desenvolvimento Web é a área da computação que ensina a criar sites e sistemas dinâmicos para a internet. Isso envolve aprender a estruturar o conteúdo das páginas com HTML, deixá-las visualmente atraentes com a estilização em CSS, e torná-las totalmente interativas utilizando a linguagem JavaScript. Estas são, exatamente, as três tecnologias essenciais usadas para construir este próprio site que você está explorando agora.

Ao longo das próximas páginas, você vai entender melhor o que é, na prática, essa disciplina, por que ela é tão importante no mercado atual, quais são os conteúdos estudados durante o semestre e como tudo isso se aplica no mundo real.

Além de dominar essas três tecnologias básicas, o desenvolvedor web moderno também entende de responsividade, otimização de performance e boas práticas de Interface e Experiência do Usuário (UI/UX). A evolução constante da internet exige que os profissionais estejam sempre atualizados com os novos padrões de mercado e novos frameworks.

O que você vai encontrar neste site

1. O que é a disciplina de Desenvolvimento Web

Uma introdução completa sobre os fundamentos da criação para a internet. Abordamos os conceitos básicos de como os navegadores interpretam o código e o papel fundamental de cada linguagem na construção estrutural e visual de uma página web moderna.

2. Por que ela é importante para quem estuda ADS

Exploramos o impacto da web no mercado de tecnologia atual. Discutimos a alta empregabilidade, as exigências da indústria de software e como o domínio dessas ferramentas é um diferencial competitivo crucial para o sucesso de um Analista e Desenvolvedor de Sistemas.

3. Os principais conteúdos estudados

Um mergulho nas tecnologias essenciais: detalhamos a tríade do desenvolvimento web (HTML, CSS e JavaScript). Além disso, explicamos a utilização de bibliotecas e frameworks (como o Bootstrap) e como planejar interfaces responsivas que se adaptam a computadores, tablets e celulares.

4. Um formulário para enviar dúvidas

Uma área interativa onde você pode testar na prática o funcionamento de um formulário de contato. Aplicamos validações lógicas utilizando JavaScript para garantir que as informações sejam preenchidas corretamente antes do envio, simulando um ambiente real de sistema.
